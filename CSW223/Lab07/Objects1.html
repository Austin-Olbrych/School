The 'jsOne.js' file contains an init function called when the html page is first loaded, 6 global variables whose names all start with '_g_', and several functions that are called when the user clicks on a button.

The 'Ball.js' file contains a Ball class. See comments in the file concerning the difference between private and public data.


To make certain that you understand the two basic ways that public data may be used, create and use the two following functions.

1. Add a function to 'jsOne.js' that uses two 'alert' statements to display the current 'over' and 'down' values of the two Ball objects by accessing the public data members from outside the class. Then call that function with a button.

2. Also, add another function in the Ball class that uses an 'alert' to display the current 'over' and 'down' values by accessing the public data members from inside the class. Then call that function twice with a button to display the 'over' and 'down' values for the two Ball objects.
